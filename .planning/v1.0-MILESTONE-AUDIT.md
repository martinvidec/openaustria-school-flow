# Milestone v1.0 — Audit Report

**Audited:** 2026-04-09

## Executive Summary

The v1.0 milestone delivers a functionally complete school management platform spanning authentication, school data modeling, automatic timetable generation, real-time editing, classbook, substitution planning, communication, homework/exams, data import, mobile PWA, and production deployment. **All 9 phases are code-complete with passing test suites (413 backend tests, 0 failures).**

However, integration audit found **3 runtime blockers** (2 from a single double-prefix bug, 1 from missing solver callback URL config) and **3 DSGVO compliance gaps** where Phase 2 services need extension to cover personal data added by Phases 5–8.

The blockers are localized and small. None require architectural changes. All can be closed in a single gap-closure phase.

---

## Findings by Severity

### Blockers (Runtime Failures)

1. **IMPORT-01, IMPORT-02 — ImportController double prefix (Finding 1)**
   - File: `apps/api/src/modules/import/import.controller.ts:43`
   - Fix: Remove `api/v1/` from `@Controller()` decorator
   - Same fix as already applied to 5 other controllers in Phase 5 Plan 10

2. **TIME-04 — Solver callback URL mismatch (Finding 2)**
   - Files: `solve.processor.ts:44`, `solver-client.service.ts:35`, docker-compose.yml
   - Fix: Set `API_INTERNAL_URL=http://api:3000/api/v1/api/internal/solver` env var

### Compliance Gaps (DSGVO)

3. **DSGVO-04 — Data export incomplete (Finding 4)**
   - File: `apps/api/src/modules/dsgvo/export/data-export.service.ts`
   - Missing tables: AttendanceRecord, GradeEntry, AbsenceExcuse, StudentNote, Message, MessageRecipient, Notification, Homework, Exam

4. **DSGVO-05 — Anonymization incomplete (Finding 6)**
   - File: `apps/api/src/modules/dsgvo/deletion/data-deletion.service.ts`
   - Missing anonymization in Phase 5–8 linked records

5. **DSGVO-06 — Retention cron incomplete (Finding 5)**
   - File: `apps/api/src/modules/dsgvo/retention/retention.service.ts`
   - Reports counts but doesn't delete grades/attendance/messages

### Feature Gaps

6. **TIME-06 — /solver Socket.IO has no frontend consumer (Finding 3)**
   - Backend emits, frontend has no listener
   - Need: createSolverSocket, useSolverSocket hook, solve trigger UI

### Minor

7. **PermissionSubject enum incomplete (Finding 9)** — Runtime OK, type gap only
8. **API_INTERNAL_URL not in .env.example (Finding 8)** — Documentation gap
9. **Stale verification truth in Phase 6 (Finding 7)** — Documentation drift
