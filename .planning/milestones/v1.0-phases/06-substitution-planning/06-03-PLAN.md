1. Export `RANKING_WEIGHTS` as a `const` object with these exact values:
   - subjectMatch: 0.45
   - fairness: 0.30
   - workloadHeadroom: 0.20
   - klassenvorstand: 0.05
   - Add a comment: `// Invariant: sum = 1.00. Researched defaults per D-05; tune by editing these values (no schema migration needed).`

2. `rankCandidates()` algorithm:
   a. Fetch all `Teacher` rows for the school where `id != absentTeacherId`, including `person`, `qualifications (TeacherSubject)`, `availabilityRules`, `reductions`.
   b. For performance (avoiding N+1), pre-fetch ALL commitments in ONE query each:
      - `timetableLessons`: `findMany({ where: { runId: activeRunId, dayOfWeek, periodNumber, weekType: { in: [weekType, 'BOTH'] }, teacherId: { in: candidateIds } } })`
      - `existingSubstitutions`: `findMany({ where: { date, periodNumber, substituteTeacherId: { in: candidateIds }, status: { in: ['PENDING', 'OFFERED', 'CONFIRMED'] } } })`
      - `roomBookings`: `findMany` on RoomBooking at the target date/period; then resolve `bookedBy` -> `Person.keycloakUserId` -> `Teacher.id` (join via Person table). Collect a Set of teacherIds with bookings at that slot.
      - `klassenvorstandOfAffectedClass`: `schoolClass.findUnique({ where: { id: classId }, select: { klassenvorstandId: true } })`
      - `givenSubstitutionsInWindow`: `groupBy` on Substitution where `substituteTeacherId IN candidateIds AND date IN [windowStart, windowEnd] AND status = 'CONFIRMED'`, select count per teacher.
      - `currentWerteinheitenPerTeacher`: use `werteinheiten.util.ts::calculateWerteinheiten` (pure fn) with TimetableLesson data — pre-compute for each candidate.
      - `werteinheitenTargetPerTeacher`: use `werteinheiten.util.ts::calculateMaxTeachingHours(teacher, reductions)` — pure fn.

   c. For each candidate, call `passesHardFilters(teacher, commitments, input)`:
      - Return false if any lesson conflict, substitution conflict, room booking conflict, BLOCKED AvailabilityRule match, MAX_HOURS_PER_DAY violation, or werteinheiten overflow.
      - Otherwise return true.

   d. For each candidate passing hard filters, call `computeScore(teacher, input, commitments)`:
      - `subjectMatch`: 1.0 if `teacher.qualifications.some(q => q.subjectId === input.subjectId)`, else 0.0 (omit the 0.5 "related Lehrverpflichtungsgruppe" rule for v1 — flag as TODO comment since the Gruppe relationship isn't in the schema yet).
      - `fairness`: `maxGiven = max(givenSubstitutionsInWindow.values())`. If `maxGiven === 0`, return 1.0. Else `1 - (candidateGiven / maxGiven)`, clamped to [0,1].
      - `workloadHeadroom`: `(target - current) / target`, clamped to [0,1]. If `target <= 0`, return 0 (defensive).
      - `klassenvorstand`: 1.0 if `klassenvorstandOfAffectedClass.klassenvorstandId === teacher.id`, else 0.0.
      - `total = 0.45*subjectMatch + 0.30*fairness + 0.20*workloadHeadroom + 0.05*klassenvorstand`.

   e. Build `RankedCandidate[]`, filter out null (hard-filter failures), sort by:
      - Primary: `b.score - a.score` (descending)
      - Tie-break: `a.teacherId.localeCompare(b.teacherId)` (deterministic per 06-RESEARCH.md)

3. Expose `passesHardFilters` as a public method so Plan 02's `SubstitutionService.assignSubstitute` can re-use it inside its Serializable transaction (Pitfall 2).

4. Expose `computeScore` as public too (for unit testability).

5. Do NOT return candidates with `score === 0` — filter them out? No, keep them per D-05 ("Admin sees the full sorted list"). Only hard-filter failures are dropped.

```typescript
import { Logger } from '@nestjs/common';
import { JwtService } from '@nestjs/jwt';
import { OnGatewayConnection, OnGatewayDisconnect, WebSocketGateway, WebSocketServer } from '@nestjs/websockets';
import { Server, Socket } from 'socket.io';
import type { NotificationDto } from '@schoolflow/shared';

@WebSocketGateway({
  namespace: 'notifications',
  cors: { origin: '*' },
  transports: ['websocket', 'polling'],
})
export class NotificationGateway implements OnGatewayConnection, OnGatewayDisconnect {
  private readonly logger = new Logger(NotificationGateway.name);

  @WebSocketServer() server!: Server;

  constructor(private readonly jwtService: JwtService) {}

  async handleConnection(client: Socket): Promise<void> {
    const token = this.extractToken(client);
    if (!token) {
      this.logger.warn(`NotificationGateway: connection ${client.id} rejected — no token`);
      client.disconnect(true);
      return;
    }
    try {
      const payload: any = await this.jwtService.verifyAsync(token);
      const userId = payload.sub;
      if (!userId) throw new Error('jwt.sub missing');
      client.join(`user:${userId}`);
      this.logger.debug(`NotificationGateway: ${client.id} joined user:${userId}`);
    } catch (err) {
      this.logger.warn(`NotificationGateway: connection ${client.id} rejected — ${(err as Error).message}`);
      client.disconnect(true);
    }
  }

  handleDisconnect(client: Socket): void {
    this.logger.debug(`NotificationGateway: ${client.id} disconnected`);
  }

  emitNewNotification(userId: string, dto: NotificationDto, unreadCount: number): void {
    this.server.to(`user:${userId}`).emit('notification:new', dto);
    this.server.to(`user:${userId}`).emit('notification:badge', { unreadCount });
  }

  emitBadgeUpdate(userId: string, unreadCount: number): void {
    this.server.to(`user:${userId}`).emit('notification:badge', { unreadCount });
  }

  private extractToken(client: Socket): string | null {
    const auth = (client.handshake as any).auth?.token ?? (client.handshake.headers as any)?.authorization;
    if (!auth) return null;
    const str = String(auth);
    return str.startsWith('Bearer ') ? str.slice(7) : str;
  }
}
```

```typescript
  @CheckPermissions({ action: 'delete', subject: 'handover' })
  @HttpCode(204)

    await this.service.deleteNote(id, req.user.id);
  }

  // Multipart endpoint — upload attachment (Pitfall 5: separate from JSON body endpoint)
  @Post(':noteId/attachments')
  @CheckPermissions({ action: 'create', subject: 'handover' })
  async uploadAttachment(@Param('noteId') noteId: string, @Req() req: any) {
    // @fastify/multipart is globally registered in main.ts with 5MB limit
    const file = await req.file();
    if (!file) throw new BadRequestException('Keine Datei hochgeladen');

    const buffer = await file.toBuffer();
    return this.service.saveAttachment(noteId, {
      filename: file.filename,
      mimeType: file.mimetype,
      buffer,
    });
  }

  @Get('attachments/:id')
  @CheckPermissions({ action: 'read', subject: 'handover' })
  async downloadAttachment(@Param('id') id: string, @Req() req: any, @Res() reply: any) {
    const { stream, attachment } = await this.service.getAttachmentStream(id, req.user.id);
    reply.header('Content-Type', attachment.mimeType);
    reply.header('Content-Disposition', `attachment; filename="${attachment.filename}"`);
    reply.send(stream);
  }

  @Delete('attachments/:id')
  @CheckPermissions({ action: 'delete', subject: 'handover' })
  @HttpCode(204)
  async deleteAttachment(@Param('id') id: string, @Req() req: any) {
    await this.service.deleteAttachment(id, req.user.id);
  }
}
```
