    **Create TeacherService** at `apps/api/src/modules/teacher/teacher.service.ts`:
    - @Injectable(), constructor(private prisma: PrismaService)
    - `async create(dto: CreateTeacherDto)`:
      - Create Person first with personType: 'TEACHER', schoolId, firstName, lastName, email, phone, address, dateOfBirth, socialSecurityNumber
      - Create Teacher linked to Person via personId, with HR fields from dto
      - Create TeacherSubject records for each subjectId in dto.subjectIds
      - Create AvailabilityRule records for each rule in dto.availabilityRules
      - Create TeachingReduction records for each reduction in dto.reductions
      - Use Prisma nested create: `prisma.person.create({ data: { ...personData, teacher: { create: { ...teacherData, qualifications: { create: [...] }, availabilityRules: { create: [...] }, reductions: { create: [...] } } } } })`
      - Return full teacher with includes
    - `async findAll(schoolId: string, pagination: PaginationQueryDto)`:
      - Filter by schoolId, include person + qualifications.subject + availabilityRules + reductions
      - Return PaginatedResponseDto with total count
    - `async findOne(id: string)`:
      - Find teacher by id with full includes
      - Throw NotFoundException if not found: `Teacher with ID ${id} was not found.`
    - `async update(id: string, dto: UpdateTeacherDto)`:
      - Find existing teacher first (404 check)
      - Update Person fields (firstName, lastName, email, phone, address, dateOfBirth, socialSecurityNumber)
      - Update Teacher HR fields
      - If dto.subjectIds provided: delete existing TeacherSubject, create new ones
      - If dto.availabilityRules provided: delete existing AvailabilityRules, create new ones
      - If dto.reductions provided: delete existing TeachingReductions, create new ones
      - Use prisma.$transaction for atomicity
    - `async remove(id: string)`:
      - Find existing (404 check)
      - Delete teacher (cascades to qualifications, rules, reductions)
      - Delete associated person record
    - `async getEffectiveCapacity(id: string)`:
      - Load teacher with reductions
      - Calculate using calculateMaxTeachingHours(teacher.werteinheitenTarget, reductions)
      - Return { werteinheitenTarget, totalReductions, effectiveWerteinheiten, maxWeeklyHours }
    - `private fullInclude()`:
      - person: true, qualifications: { include: { subject: true } }, availabilityRules: true, reductions: true
