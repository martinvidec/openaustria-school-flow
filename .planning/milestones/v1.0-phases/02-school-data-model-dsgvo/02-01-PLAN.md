    **Step 4: Create EncryptionService** at `apps/api/src/modules/dsgvo/encryption/encryption.service.ts`:
    - Injectable NestJS service
    - Uses Node.js `crypto` module: createCipheriv, createDecipheriv, randomBytes, scryptSync
    - Algorithm: AES-256-GCM
    - Constructor takes encryption key from ConfigService: `this.config.get<string>('SCHOOLFLOW_ENCRYPTION_KEY')`
    - If key is not configured, log warning but do not crash (allow app to start without encryption for dev)
    - Key derivation: `scryptSync(key, 'schoolflow-salt-v1', 32)`
    - Encrypted format: `$enc:v1:{iv_hex}:{authTag_hex}:{ciphertext_hex}`
    - Methods:
      - `encrypt(plaintext: string): string` -- returns encrypted string with $enc:v1: prefix
      - `decrypt(ciphertext: string): string` -- returns plaintext, passes through non-prefixed strings unchanged
      - `isEncrypted(value: string): boolean` -- checks for $enc:v1: prefix
      - `isConfigured(): boolean` -- returns true if encryption key is set
    - IV: 16 random bytes per encryption (non-deterministic, per D-16 design)
