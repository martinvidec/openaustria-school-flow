   - useUploadImport(schoolId): useMutation. mutationFn uploads file via FormData to POST /api/v1/schools/{schoolId}/import/upload. Returns parse result (entity counts or CSV headers+rows).
   - useDryRun(schoolId): useMutation. mutationFn POSTs to /import/dry-run with StartImportRequest body. Returns ImportJobDto.
   - useCommitImport(schoolId): useMutation. mutationFn POSTs to /import/{importJobId}/commit. Returns ImportJobDto.
   - useImportHistory(schoolId): useQuery. queryKey ['import-history', schoolId]. Fetches GET /import/. Returns ImportJobDto[].
